Урок 13

Задание 3

С использованием замыканий разработать итератор, то есть функцию, которая будет постепенно по отдельным вызовам выдавать по одному значению от заданного минимального до заданного максимального. Если значение достигнет максимального, следующим будет минимальное значение. При использовании этой функции реализовать перебор номеров месяцев
